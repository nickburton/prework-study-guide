Prework Study Guide

HTML

  • HTML adds the frame for a page.
  • HTML pages have a HEAD tag for metadata and a BODY tag for human visible content.

CSS

  • CSS adds styling to an HTML page
  • CSS can be added to types, classes, or IDs of of HTML elements
  • Margin adds space outside of an element
  • Padding adds space inside of an element, like a padded room

Git

  • 'git status' checks what branch we are currently on
  • 'git checkout -b branch-name' creates a new branch and switches to it
  • 'git add .' adds all files to the staging area
